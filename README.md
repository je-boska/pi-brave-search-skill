# brave-search

Pi skill for searching the web with the Brave Search API and returning compact, agent-friendly results.

## Install

```bash
pi install git@github.com:je-boska/pi-brave-search-skill.git
```

Then run `/reload` in Pi if already running.

You need a Brave Search API key.

Store it at `~/.config/pi/brave-search-token.json`:

```json
{ "apiKey": "..." }
```
